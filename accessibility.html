Accessibility Statement

Our commitment to making LTC Market Place accessible to everyone

Our Commitment

LTC Market Place is committed to ensuring digital accessibility for all users, including people with disabilities. We continually improve the user experience for everyone and apply the relevant accessibility standards throughout our platform.

As a marketplace built around equality for disabled people, accessibility isn't an afterthought — it's foundational to everything we do.

Standards We Follow

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA. These guidelines explain how to make web content more accessible to people with disabilities.

Perceivable

  • Text alternatives for all non-text content
  • Captions and alternatives for multimedia
  • Content can be presented in different ways
  • Content is distinguishable with sufficient contrast

Operable

  • Full keyboard navigation support
  • Skip navigation links
  • Sufficient time to read and use content
  • Clear focus indicators on all interactive elements

Understandable

  • Clear, readable text content
  • Predictable navigation and operation
  • Input assistance and error prevention
  • Consistent interface patterns

Robust

  • Compatible with assistive technologies
  • Valid, semantic HTML markup
  • ARIA labels and roles where appropriate
  • Progressive enhancement approach

Accessibility Features

  • ✓ Skip to main content link on every page
  • ✓ Semantic HTML5 landmarks and structure
  • ✓ ARIA labels on all interactive elements
  • ✓ Keyboard-navigable throughout the entire site
  • ✓ High contrast colour ratios meeting AA standards
  • ✓ Responsive design for all screen sizes
  • ✓ Focus indicators on all buttons and links
  • ✓ Alternative text on all images
  • ✓ Form labels and error messages linked to inputs
  • ✓ Live regions for dynamic content updates
